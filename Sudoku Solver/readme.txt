################################################################
Put your sudoku puzzle in the input file.
The file must contain 9 rows, 9 characters each.
The characters must be either:
	- digits from 1 to 9 only,
	- or a space character if it's to be empty at start.
Rows need to be filled up to the last, 9th character.

Then run the .exe file and your solution will appear on screen.
